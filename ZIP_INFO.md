# Архив для скачивания

Если бинарный `.zip` не отображается в интерфейсе, используйте текстовый файл:
`orksky-rubezh-game.zip.base64.txt`.

## Как восстановить ZIP

```bash
base64 -d orksky-rubezh-game.zip.base64.txt > orksky-rubezh-game.zip
```

После этого получите обычный архив `orksky-rubezh-game.zip`.
